# Pipeline Base Image

This repository contains the Dockerfile for building a base image used in pipelines.

## Included Tools and Dependencies

- **Node.js**: `lts-alpine3.17`
- **Bash**: `5.1.16-r2`
- **Sudo**: `1.9.12_p2-r0`
- **Shadow**: `4.13-r2`
- **JQ**: `1.6-r1`
- **Trivy**: `v0.45.0`
- **Helm**: `v3.13.0`
- **Kubectl**: `v1.28.5`
